BBAROLO

A 3D fitting tool for the kinematics of galaxies

Barolo (Best-fit Analysis of Rotating Object from Line Observations) is a tool for fitting 3D tilted-ring models to emission-line datacubes. Bbarolo works with 3D FITS files, i.e. image arrays with two spatial and one spectral dimension.

## Pre-requisites
Bbarolo compiles and runs on Unix machines, supporting Mac OS X and any Linux distros. It just need a C++ compiler, like the GNU compiler (better if \> 4.2), the CFITSIO and the FFTW libraries. You can easily install them through the terminal commands of the various package managers, i.e. `apt-get` on Ubuntu-based, `pacman` on Arch-based, `yum` on RPM-based distros, `brew` or `port` on Mac OS X.


## Installation

1) Download either the .zip or the .tar source file.

2) Uncompress it (`unzip *filename*.zip` or `tar -xvzf *filename*.tar`) and enter the Bbarolo directory in a terminal prompt.

3) Launch the autoconfigure script (`./configure`). If there are some errors, follow the suggestions given by the script for manually set the path of libraries.

4) Compile the source by typing `make`.

5) Clean up the unnecessary files (`make clean`).
