Barolo is available for Mac OS X and Linux platforms. The code run as a command line program, but we have also implemented a simple Graphical User Interface (GUI) as a front-end for the input parameters.

For downloading the pre-compiled binaries click here (additional libraries not needed, but still need Gnuplot and Astropy for the output plots).

For compiling Barolo from the source, click here.

Examples can be found here.


NOTE: Barolo needs writing permissions to run, thus do not put the executable in root directories!
